# Ash Nazg documentation

Ash Nazg is a Nextcloud ExApp that lets users upload arbitrary binaries to
their Files and execute them inside sandboxed runtime engines, with output
streamed to the browser and writes flowing back into Files via WebDAV. See
the [project README](../README.md) for the full overview, motivation, and
current status (alpha — scaffolding, not for production).
